chatter
=======

Another node.js/socket.io chat example

Requirements
--------------

- node v0.10.18
- npm 1.3.8

Installation
--------------

Clone github repository
```bash
git clone https://github.com/dieg0v/chatter.git
```
Install dependencies
```bash
npm install
```
Run app
```bash
node app
```
Open your browser
```bash
http://127.0.0.1:3000/
```

Feel free to ask any questions
